Product Overview

Product Vision

NebloBot is an AI-powered communication and execution agent that integrates with both broker/carrier TMS platforms and messaging platforms like WhatsApp and Telegram. It acts as a digital dispatcher, automating load matching, capacity discovery, booking, and negotiation in real time.

The goal is not to replace TMS platforms, but to sit on top of them, acting as a smart layer that automates communication and executes freight actions, where brokers and carriers already operate â€” in chat apps.

Key Problems Addressed

  1. Brokers working across 30-40 Telegram/WhatsApp groups to manage loads daily
  2. Lost load opportunities due to slow replies, message overload, or missed posts
  3. Poor carrier engagement with broker portals (TMS portals)
  4. Time-consuming manual communication and data entry for brokers and dispatchers

Core Features (MVP)

  1. Messaging Platform Integration
  2. Telegram Bot
  3. WhatsApp Business API (via Twilio or Gupshup)

  4. Real-time message processing

  5. AI Message Parsing Engine

  6. Intent extraction from natural language

  7. Message classification (load request, bid, availability, quote, status update)

  8. Load Matching Logic

  9. Match carrier capacity with uncovered loads

  10. Bid/accept/decline actions through the bot

  11. TMS Integration Adapter

  12. Pull: Active loads, uncovered loads, carrier database

  13. Push: Bookings, bids, status updates

  14. AI Agents for Specific Tasks

  15. BookingBot, ScoutBot, ReminderBot

Technical Architecture

  1. Chat Layer: WhatsApp (Twilio), Telegram Bot API
  2. NLP/LLM Layer: GPT-4 or Claude for message understanding & response generation
  3. TMS Adapter Layer: REST/Webhook integration with broker TMS systems
  4. Logic Orchestrator: LangChain/Semantic Kernel agent logic for workflows
  5. Vector DB: Pinecone or Qdrant for matching history and personalization
  6. Database: Stores structured user/load/interaction data

Success Metrics

  • Loads matched via NebloBot: 50+/week
  • Average broker response time reduction: -40%
  • AI engagement rate per load: 60%+
  • Load coverage success rate: +30% improvement
  • Reduction in manual data entry: -50%
